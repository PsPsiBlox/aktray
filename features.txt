Player:
	- Walk Speed/JumpPower:
		Master Switch - enables them
		Value - value of speed/jumppower
		Disable If Handcuffed - disables them if handcuffed
	   *⚠️ These features are detected. To use them you have to press "Yes" on the confirmation GUI
	- Utils:
		Anti Ragdoll - disables ragdoll
		Anti Skydive - disables skydive (do NOT enable while skydiving)
		Anti Parachute - disables parachute
		Anti Falldamage - disables falldamage
		Anti Taze - disables taze ragdoll
		Parachute On Jump - parachutes on jump
		Delay - delay for parachute on jump (in milliseconds)
		Parachute Boost - modifies parachutes to be able to "fly"
		Jesus - allows you to walk/drive on water
		Choose Respawn - kills character to receive "Choose Spawn" GUI
	- Misc:
		FOV - field of view
		Infinite Duck - lets you to spam crawling
		Infinite Punch - lets you to punch faster
		Always Jump - lets you jump while holding spacebar
		Always Sprint - allows you to always sprint without pressing SHIFT
		Always Punch - spams punch
		Always Duck - spams duck
		Automatic Respawn On Stunned - automatically changes you to team prisoner when tazed/c4 explodes near you
		Allow Equip On Duck - lets you equip items while crawling
		Allow Equip While Flying - lets you equip items while flying a jetpack
		Allow Equip With Items - lets you unequip items that are locked
Combat:
	- Gun Mods:
		Automatic Fire - makes weapons automatic
		No Recoil - removes weapons recoil
		Anti Knockback - removes knockback from flintlock weapon
		Instant Reload - attempts to instantly reload your weapon by reequipping

		Open Gunstore UI - opens gunstore gui

	- Silent Aim:
		Master Switch - enables silent aim
		FOV - field of view
		FOV Circle - circle
		Movement Prediction - attempts to predict next movement of your target
		Prediction Value - value of prediction
			* You can modify this until you hit flying/fast moving players
		Allow Target Prisoner - allows aimbot to target prisoners
		Allow Target NPCs - allows aimbot to target npcs
		Allow Target Boss - allows aimbot to target Boss from Mansion Robbery
		Head Only - allows aimbot to shoot Boss only in the head to deal the most amount of damage
		Penetrate Walls - allows aimbot to shoot through walls
		Automatic Shoot - automatically shoots with silent aim (penetrate walls needs to be enabled for now)
	- Arrest Aura:
		Master Switch - enables arrest aura
		Range - range of arrest aura
		Automatic Equip Handcuffs - equips handcuffs automatically
		Automatic Eject Player - ejects player out of vehicle automatically
		Allow Tase Target - allows arrest aura to tase targets before arresting
		Allow Through Walls - allows arrest aura to arrest through walls
		Talk on Arrest - advertises the script discord server when arresting somebody
	   *⚠️ You have to be on the Police team
